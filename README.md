# 🎫 Conference Ticket Generator

A modern frontend project that allows users to generate a personalized conference ticket by uploading an avatar, filling out a form, and validating their information. The application includes drag-and-drop image upload, real-time validation, and dynamic ticket generation with smooth UI animations.

---

* 💻 GitHub Repository: https://github.com/shadymo2291/JavaScript-Projects-Conference-Ticket-GeneratorSolution

## 🚀 Features

* 🎨 Modern responsive UI design
* 🖼️ Drag & Drop image upload
* 📁 File validation (type & size restrictions)
* 🧠 Real-time form validation
* 📧 Email validation using regular expressions
* 🎫 Dynamic ticket generation
* 📅 Auto-formatted date display
* 👤 Live preview of user data on ticket
* ❌ Image remove & replace functionality
* ✨ Animated ticket appearance for better user experience

## 📸 How It Works

1. Upload your avatar (drag & drop or click)
2. Fill in your full name, email, and GitHub username
3. Validate inputs in real time
4. Click **"Generate My Ticket"**
5. View your animated personalized conference ticket 🎟️
